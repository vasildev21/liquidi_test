## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

Use the NestJS CLI to create a new project: nest new instagram-like-poc
Install required dependencies: npm install (or yarn install)
User Model:

Create a user.entity.ts file to define a user model with properties like:
id
username (or email)
password (hashed)
instagramId (optional, to store user's Instagram ID)
Authentication (Optional):

Implement basic user registration and login functionalities (consider using Passport.js with Facebook strategy)
Store user credentials securely after hashing the password.
Facebook Integration:

Register your application with Facebook Developer Platform.
Use the Facebook SDK for Node.js to handle authorization flow.
Upon successful authorization, retrieve the user's Instagram ID (if available) and store it in the user model.
Instagram Like Tracking:

Utilize the Instagram Graph API to access information about the designated Instagram account (likes count, etc.).
Important Note: Retrieving individual user likes on another account might be restricted by Instagram's API. We might need to rely on the user's self-reported likes through the app (discussed later).
Alternatively, explore Instagram Insights API (for Business/Creator accounts) if applicable to your use case.
Like Counter:

Implement a service/controller to track user likes on the monitored Instagram account.
This could involve:
Storing the monitored account ID in a configuration file.
Allowing the operator to change the monitored account easily.
(Optional) User interface to display the user's like count for the monitored account (consider a simple counter component).
